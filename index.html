Tarefa: Para pegar um ônibus

Entrada: Localizar o ponto de embarque, Chegar na hora.

Processamento:

  • Achar rota onde o ônibus passa.
  • Consultar os horários de chegado e partida.
  • Leve dinheiro para pagar o bilhete.
  • após localizando a rota do ônibus. Chegue ao ponto de ônibus com algum minutos de antecedência.
  • Consulte o letreiro (que fica penduradas no parabrisas dos veículos) e vejo se está no ponto certo.
  • Espere os passageiros descerem antes de subir no ônibus.
  • Pague o bilhete
  • Senta no banco de passageiro do ônibus.

Saida: Espere chegar no local que quer visitar.

Tarefa: Trocar o pneu

Entrada: Ligar para o borracheiro, sair do carro, usar o macaco, Chave Parafusadeira

  • se o estepe está vazio então: chamar borracheiro
  • se não
  • Saia do carro
  • usando o macaco, levantar o carro
  • Com a chave parafusadeira desapertar o parafuso, enquanto houver parafuso faça o mesmo processo
  • desparafusar a roda
  • fim enquanto
  • remover a roda
  • colocar o estepe
  • Com a chave parafusadeira apertar o parafuso, enquanto houver parafuso faça o mesmo processo
  • parafusar a roda

    • Saida: Entre no carro e volte para estrada
